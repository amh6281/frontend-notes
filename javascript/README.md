# JavaScript

프론트엔드 개발에 필요한 JavaScript 핵심 개념을 정리합니다.

## 학습 순서

1. Data Types
2. Equality
3. Function
4. Class
5. Closure
6. Event Loop
7. Syntax
8. TypeScript

## 문서

| Topic      | Description                               |
| ---------- | ----------------------------------------- |
| Data Types | 원시 타입과 객체 타입, 데이터 표현 방식   |
| Equality   | `==`, `===`, `Object.is`, 얕은 비교       |
| Function   | 함수 선언, 표현식, 화살표 함수, 일급 객체 |
| Class      | 클래스, 프로토타입, 상속                  |
| Closure    | 렉시컬 환경과 클로저                      |
| Event Loop | 이벤트 루프와 비동기 처리                 |
| Syntax     | 구조 분해, 전개 연산자 등 주요 문법       |
| TypeScript | React에서 자주 사용하는 TypeScript 개념   |
